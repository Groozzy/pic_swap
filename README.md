# pic_swap
Данный репозиторий представляет собой попытку создания API с использованием Python библиотеки Falcon. Целью данного проекта является предоставление простого и удобного инструмента для разработки RESTful API на языке Python. Репозиторий содержит примеры кода, которые демонстрируют основные возможности Falcon.
